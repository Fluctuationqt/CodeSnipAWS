Относно AWS услугите срещнахме проблеми с CORS policy при повикване на API Gateway ресурс от Ajax заявките изпратени от фронтедна.
Това се случваше, защото AWS изисква всеки response от изпълнението на ламбда функциите да съдържа
"Access-Control-Allow-Origin: /*domain here*/", въпреки че самият API Gateway ресурс има настроено CORS policy,
което позволява това. Също така при използването на DynamoDB имаше нужда да търсим данни в таблица по индекс, който не беше задеден за Primary Key,
коригирахме таблицата си, като и добавихме Sort Key и използвахме двата ключа съвместно при търсене.
